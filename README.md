# Vasu Dev - Experimental Portfolio

**Live URL** - [https://vasu-dev-portfolio-d7lf.vercel.app/](https://vasu-dev-portfolio-d7lf.vercel.app/)

This isn’t the typical recruiter friendly portfolio. This is an experimental portfolio (creativity over convention).

I’ve always found most portfolios to be repetitive and template-driven — so instead of following that path, I built this to push my limits as a frontend developer.

I feel like a portfolio (especially as a frontend developer) shouldn’t just tell people you’re good, it should make them feel it.

## Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion, GSAP
- **Smooth Scroll:** Lenis
- **Routing:** React Router v6

## Studio Archive / Shipped Products

This repository includes custom case studies for the following shipped products, built using a dynamic routing engine with smooth animated page transitions:

1.  **SmartEdLabs** - AI-powered language learning platform featuring Super Admin, Student, Tutor, and Institute modules. (Next.js, TypeScript, Node.js, Express, MongoDB, Tailwind, Groq AI)
2.  **Amata Farms** - A modern e-commerce platform for premium desi ghee, delivered online with secure Razorpay payments. (Next.js, TypeScript, TurboRepo, Tailwind, Razorpay)
3.  **Nourished** - A smart food delivery platform ensuring healthy meals for children with QR code scanning for secure handoffs. (Next.js, TypeScript, Node.js, Express, MongoDB, QR Code)
4.  **PITCS** - A corporate website showcasing IT consulting, staffing, and digital transformation services. (Next.js, TypeScript, Tailwind, Framer Motion, GSAP)
5.  **Travelum** - A travel blogging platform featuring destination guides and travel stories with SEO optimization. (Next.js, TypeScript, Node.js, MongoDB, Tailwind)
6.  **Social Notch** - An event management platform for creating unforgettable celebrations and corporate events. (Next.js, TypeScript, Tailwind, Framer Motion, GSAP)
7.  **PhoneNest** - A trusted mobile repair service platform with device diagnostics and customer support booking. (Next.js, TypeScript, Node.js, MongoDB, Tailwind)

_Also includes detailed, standalone custom case study implementations for **Elevate** and **Zentra**._

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   # or
   pnpm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
